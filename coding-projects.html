Mo Money

C++

Course: ECE 150: Fundamentals of Programming

The "Mo Money" project is an advanced C++ application designed to manage and process transactions for an investment firm specializing in Exchange-Traded Funds (ETFs). The project's primary objective is to implement an Adjusted Cost Base (ACB) reporting system to accurately compute capital gains and losses for tax reporting to the Canada Revenue Agency (CRA).

Project Breakdown:
Data Structures and Class Design:

  • Transaction Class: Encapsulates transaction details (ticker, date, type, shares, value) and manages the linked list structure with pointers. Includes constructors, destructors, and overloaded operators for sorting and management.
  • History Class: Controls the transaction linked list, handling creation, sorting, updating, and printing. Manages dynamic memory and provides methods for inserting, sorting by date, updating ACB/CGL, and computing capital gains/losses.
Memory Management: Emphasizes dynamic memory management through pointers, with careful allocation and deallocation to prevent leaks.
File I/O and Data Parsing: Reads transaction data from an external file, parses it, and creates Transaction objects for each entry. Outputs transaction history and financial calculations.
Algorithm Implementation: Custom sorting, ACB and CGL calculations, and operator overloading for efficient management.
Technical Focus: Pointer arithmetic, class design, algorithm implementation, and operator overloading.

Outcome: A fully functional system for managing ETF transactions, including accurate ACB and CGL reporting, reinforcing advanced C++ programming skills and real-world financial problem-solving.
